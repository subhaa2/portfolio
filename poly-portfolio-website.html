Course project note: this work was completed during Year 1 Semester 2 as part of diploma coursework connected to the UI/UX and Web Development modules.

I completed this portfolio website during my second semester of Year 1 in my diploma program. The project connected two of my modules, UI/UX and Web Development, so it involved both visual planning and implementation.

The main objective was to first create a visual representation of the website in Illustrator and then translate that design into a working website using HTML and CSS. The result was a portfolio site with dedicated sections such as the homepage, About Me page, portfolio gallery, and contact page.
