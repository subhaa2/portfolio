J.P. Morgan Internship

Trade Status Website

A client-facing website that made booked deal information easier to access through end-of-day and intraday reporting views.

Confidentiality note: interface visuals and internal implementation details are simplified here for portfolio use, but the workflow and responsibilities reflect the real internship project.

This project focused on making trade information more accessible to clients without requiring them to manually query databases or inspect raw deal records. The website consolidated key trade details into a cleaner reporting workflow that was faster to review and easier to navigate.

I worked with Python and company-provided user interface code, then extended the experience with HTML and CSS so the reporting views were clearer and more useful for day-to-day client usage.

Python HTML CSS Reporting UI
Period
Mar - Aug 2023
Role
Software Engineer Intern
J.P. Morgan internship project focused on client reporting and usability.
Core Outcome
Faster deal visibility
Users could view booked deals through structured report views instead of checking data manually.

What it solved

  • Helped clients view information about the deals they had booked from a single web interface.
  • Reduced the need to manually go into databases and inspect each deal individually.
  • Separated reporting into end-of-day and intraday views so users could check both summary and recent activity quickly.

Key features

  • Date picker and report selection controls for switching between end-of-day and intraday views.
  • Dropdown filters and grid-based data presentation for faster scanning of booked deal information.
  • Embedded links, charts, and structured table views to support more informed follow-up actions.

Technical contribution

The website was built with Python and company-provided UI code, then refined with HTML and CSS to create a cleaner reporting experience. My focus was on integrating the data-driven reporting logic with a user interface that felt more usable for business workflows.
